Window Screen The window.screen object contains information about the user's screen. The window.screen object can be written without the window prefix. Properties: screen.width screen.height screen.availWidth screen.availHeight screen.colorDepth screen.pixelDepth The screen.colorDepth property returns the number of bits used to display one color. The screen.pixelDepth property returns the pixel depth of the screen. For modern computers, Color Depth and Pixel Depth are equal.
